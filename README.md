# STAT440 Group C Final Project

## Project Overview
This repository contains the **final project for STAT440 – Business Statistics**, Group C. The project involves analyzing multiple economic datasets to extract meaningful insights, identify trends, and provide data-driven recommendations for business decision-making.

The datasets cover the following sectors:  
- 12-Year Gasoline  
- Agricultural  
- Consumer Food  
- Energy  
- Financial  
- International Labor  
- Manufacturer  
- U.S. International Stock Market  

## Objectives
- Explore each dataset individually  
- Clean and preprocess data as needed  
- Perform statistical analysis and visualizations  
- Extract insights to support business decision-making

## How to Collaborate
1. **Fork or clone the repository** to your local machine.  
2. Create a **branch** for your work (e.g., `gasoline-analysis`).  
3. Add your scripts, visualizations, and cleaned datasets to the appropriate folder.  
4. Commit your changes and **push to the main repo**.  
5. Open a **pull request** if you want your changes merged or reviewed.  

> **Tip:** Treat GitHub as a shared workspace where everyone contributes and tracks updates.
